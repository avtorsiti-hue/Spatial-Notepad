# Spatial Notepad (Пространственный блокнот)

## Описание
Spatial Notepad — это мощное приложение для пространственного ведения заметок и организации идей на бесконечном холсте. Оно сочетает в себе эстетику "стеклянного" дизайна (glassmorphism), удобство интеллект-карт и возможности искусственного интеллекта Gemini.

## Основные возможности
- **Бесконечный холст**: Свободное перемещение, масштабирование и организация заметок в любом направлении.
- **Стеклянные ноды (Glass Nodes)**: Заметки с эффектом размытия, настраиваемыми шрифтами, цветами и выравниванием текста.
- **Визуальный редактор (Tiptap/ProseMirror)**: Полноценный Rich Text редактор (жирный, курсив, списки, заголовки, цвета).
- **Мультимедиа**: Вставка изображений и видео прямо в заметки.
- **Интеграция с Gemini AI**: Встроенный чат для генерации идей. Ответы можно мгновенно превращать в новые заметки. Поддерживает анализ изображений.
- **Темы и фоны**: Темная тема "Obsidian" с поддержкой установки собственных изображений или даже HTML-файлов в качестве живого фона.
- **Кастомные ссылки**: Возможность добавить свои часто используемые ссылки прямо в боковое меню.
- **Поиск**: Быстрый поиск по заголовкам и содержимому всех заметок на холсте.
- **История (Undo/Redo)**: Глобальная отмена и повтор действий (до 50 шагов).
- **Экспорт**: Сохранение всей вашей карты в виде красивого автономного HTML-файла.
- **PWA (Установка)**: Приложение можно установить на компьютер или телефон как отдельную программу.

## Управление и горячие клавиши
- **CTRL (Selection Mode)**: Переключает режим выделения. В активном режиме можно выделять несколько нод рамкой. В выключенном — перемещать холст.
- **Добавление заметки**: Кнопка "+" в нижнем меню или кнопка "Добавить приложение" (для установки PWA).
- **Связи**: Перетаскивайте кружки (handles) по краям заметок для создания соединительных линий.
- **Сетка и привязка**: В меню можно включить визуальную сетку и привязку нод к ней для идеального выравнивания.
- **Масштабирование**: Колесико мыши или жесты тачпада.
- **Редактирование**: Двойной клик по тексту заметки открывает редактор.

## Как установить приложение (PWA)
Нажмите кнопку **"ДОБАВИТЬ ПРИЛОЖЕНИЕ"** в левом верхнем углу экрана. Это создаст ярлык на вашем рабочем столе, и приложение будет открываться в отдельном окне без интерфейса браузера.

## Технологии
Приложение использует React, React Flow, Zustand для управления состоянием и IndexedDB для надежного локального хранения всех ваших данных.
